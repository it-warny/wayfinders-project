Descrição do Projeto

Este projeto nasceu como um presente: uma celebração digital da nossa jornada e da nossa amizade. O WayFinders é a nossa "estrada do tempo", um diário compartilhado e eterno, construído para que nós quatro possamos registrar cada parada, cada risada e cada momento que define a nossa história. É um espaço vivo, que cresce a cada nova memória adicionada, garantindo que, não importa para onde a vida nos leve, sempre teremos o caminho de volta para as lembranças que nos unem.

Sobre o Warny Palhares

Com uma carreira de duas décadas em gestão, liderei equipes e otimizei operações na aviação e hotelaria, do Brasil à Europa. Hoje, movido por uma nova paixão, estou em transição para o universo da Inteligência Artificial. Este projeto é um reflexo dessa jornada: um desafio prático onde aplico meus estudos em Python e Machine Learning para construir algo significativo. Meu objetivo é simples: combinar a experiência em resolver problemas complexos com as ferramentas da ciência de dados para criar um impacto real e inovador.
